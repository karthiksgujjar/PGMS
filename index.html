How to use?

  1. Make sure 'MySQL Server' Latest Version is installed
    • Remember the password given during MySQL installation (By default, Username: root)
    • Server Name will be localhost or 127.0.0.1
  2. Install the application using PGMS Installer.
  3. Enter the database details you've used while installing the MySQL server.
    • Enter the Server Name (localhost), User Name (root) and Password (Given during MySQL installation) of MySQL Server
  4. Enter the Details of the PG, Owner details (Only on First Run).
  5. Enter the login details and use the application.

About

PGMS (Paying Guests Management System) is a guest management software which is made on the owner's perspective. It provides a user friendly interface, using this the owner of the paying guest will be able to manage the information of the paying guests staying in their building. Paying guest owner can admits the guest by getting all the required information of the guest and he can add the Building details

This application allows the PG managing department to manages the Guests instead of physical ledgers, it computerizes the work and eases the productive time of the owner. It allows the administrator/owner to:

  • Add/Remove Buildings, Floors, Rooms
  • Add/Remove (Maximum 2 Staffs
  • Admit/Remove Guests
  • Overview the PG
  • Accept Monthly Fees
  • Keep track of Visitors
  • View and Generate Report on Guests of selected Building
